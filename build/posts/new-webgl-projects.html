New WebGL Projects

Published inwebgljavascriptweb development

October 11, 2014 by Lucas Doyle

I’ve been working on some new WebGL related projects for a 3d viewer and editor that targets robotics applications.

three.js

All these projects are built on top of three.js. My fork added kinematics support to the collada loader. Check it out!

Marionette-three.js

This is a library that wraps three.js components for Marionette. Provides a comfortable, event driven interface that abstracts away a lot of three.js things in an MVC friendly way. For example, a Drawable is a Backbone.Model instance that maintains its own three.js scenegraph node. Pass a Backbone.Collection of drawables to a three.jsRenderer view and you’re off to the races! The library also includes support for controls and loading different types of meshes inside Drawable instances. Check out an example here and code for it here.

Xaphoon

This is a small heroku app that explores collaborative editing of a 3D environment. It incoporates the previous two projects plus Backbone.IO (which I forked and added socket.io 1.0 support).
