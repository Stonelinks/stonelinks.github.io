<p>A beginner friendly, minimalistic and open source robot arm simulator I wrote in python for a robotics class back in 2012. Stoolbotics is designed to be a general purpose arm and kinematics simulator aimed at being a teaching tool for aspiring roboticists.</p>
<iframe src="http://www.youtube.com/embed/h3Rus5mUkzY" frameborder="0" allowfullscreen></iframe>
<p><a href="/projects/stoolbotics/">Home</a>

|
<a href="/projects/stoolbotics/quickstart/">Quickstart</a>

<a href="/projects/stoolbotics/docs/">Docs</a></p>
<p>Linear algebra and mathematical concepts behind robotics are difficult for a beginners without any visual context. This is especially true for people who are primarily visual learners (like me). This tool will hopefully fill a gap in education environments where robotics is being taught.</p>
<p><img src="/stoolbotics/omni.jpg" alt="Omni"></p>
<p>Stoolbotics (right) simulating a phantom omni (left).</p>
<h2>Features</h2>
<p>Stoolbotics has many features that make it attractive to the aspiring roboticist and the robotics education classroom:</p>
<ul>
<li>An easy to use / understand file format for specifying a robot arm</li>
<li>Ability to visualize any robot that can be specified in such a file</li>
<li>Compute the forward kinematics of any robot from DH parameters</li>
<li>Command line interface within simulator with many useful commands</li>
<li>Customizable simulation environment
<ul>
<li>Animate and draw paths for arms</li>
<li>Control time-stepping</li>
<li>Change variables in the simulator on the fly</li>
</ul>
</li>
<li>Record / playback simulator activity</li>
<li>MATLAB compatible
<ul>
<li>Import a recording generated in MATLAB</li>
<li>Able drive simulator in real time from a UDP stream (includes an example of this)</li>
</ul>
</li>
<li>Built in help from simulator command line and comprehensive documentation</li>
<li>Cross platform implementation</li>
</ul>
